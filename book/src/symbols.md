# Symbols

| Symbol | Description |
| ------ | ----------- |
| \\[ k \\] | private key |
| \\[ K \\] | public key |
| \\[ K.x \\] | x coordinate of the public key |
| \\[ K.y \\] | y coordinate of the public key |
| \\[ r \\] | r value of the signature |
| \\[ s \\] | s value of the signature |
| \\[ z \\] | hash of the message |
| \\[ m \\] | message |
